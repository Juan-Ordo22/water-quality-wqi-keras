# water-quality-wqi-keras
Análisis y predicción del índice de calidad del agua WQI usando Python, PySpark y Keras. El proyecto incluye limpieza e imputación de datos, cálculo de puntuaciones de calidad, visualización geográfica por estados de India y entrenamiento de un modelo de regresión para estimar el WQI a partir de variables fisicoquímicas y microbiológicas del agua.
